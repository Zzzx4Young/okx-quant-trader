## 术语约定

| 术语 | 说明 |
|------|------|
| `EMA20` | 20 周期指数移动平均线 |
| `close` | K 线收盘价 |
| `volume` | K 线成交量 |
| `long` | 做多（买入开多） |
| `short` | 做空（卖出开空） |
| `TP` | Take Profit，止盈价 |
| `SL` | Stop Loss，止损价 |
| `K线走完` | 指当前时间已超过最近一根 15m K 线的闭合时间 |
| `有效突破` | 收盘价连续 2 根 K 线站稳在 EMA20 同一侧 |

## 策略 A：EMA20 均线突破（已激活）

### 触发条件

#### 做多信号（LONG）
1. 当前无持仓或当前持仓方向为 `short`
2. **趋势确认**：最近 2 根 15m K 线收盘价 **连续高于** EMA20
3. **信号对齐**：ema_direction 由负转正（EMA20 开始上翘）
4. **量价配合**：信号 K 线成交量 ≥ 前 3 根平均成交量的 1.2 倍

#### 做空信号（SHORT）
1. 当前无持仓或当前持仓方向为 `long`
2. **趋势确认**：最近 2 根 15m K 线收盘价 **连续低于** EMA20
3. **信号对齐**：ema_direction 由正转负（EMA20 开始下拐）
4. **量价配合**：信号 K 线成交量 ≥ 前 3 根平均成交量的 1.2 倍

### 过滤条件（任一满足则跳过本次信号）

| 条件 | 说明 |
|------|------|
| `K线未走完` | 当前时间不在 15m 整数倍（00/15/30/45分）之后 |
| `已触发日内3次止损` | 系统已连续触发 3 次硬止损，进入熔断 |
| `emergency_stop=true` | 人工熔断开关已开启 |
| `已达最大持仓上限` | 当前持仓数量 ≥ `max_concurrent_positions` |

### 开仓规则

| 参数 | 值 |
|------|---|
| K 线周期 | 15 分钟（15m） |
| 交易对 | BTCUSDT / ETHUSDT |
| 杠杆 | BTC/ETH 默认 5x，上限 10x |
| 仓位模式 | 逐仓（isolated） |
| 止盈 | SL 距离 × 1.5 倍（盈亏比 ≥ 1.5） |
| 止损 | 固定为信号 K 线最低/最高 ±0.5% |
| 最大同时持仓 | 3 个交易对 |

### 平仓规则

#### 主动止盈（TP）
- 价格触及 TP 触发价 → 市价平仓 50%
- 剩余仓位 TP 上移至盈亏平衡点（break-even）
- 剩余 50% 持仓追踪移动止损

#### 主动止损（SL）
- 价格触及 SL 触发价 → 市价全平
- 计入当日连续亏损计数

#### 被动离场（趋势反转）
- 持仓为 `long`，EMA20 连续 2 根 K 线收盘价 < EMA20 → 平多
- 持仓为 `short`，EMA20 连续 2 根 K 线收盘价 > EMA20 → 平空

#### 时间止损
- 持仓超过 **2 小时** 且盈亏比未达到 1:1 → 强制市价平仓

## 策略 B：布林带 + RSI（预留，暂不激活）

> 接口已预留，激活时间待定。

### 触发条件

#### 做空信号
1. RSI(14) > 70（超买）
2. 收盘价刺破布林带上轨（close > UpperBand）
3. 出现反转 K 线形态（流星线 / 看跌吞没）
4. 做空，SL = 入场价 + 1%，TP = 布林带中轨

#### 做多信号
1. RSI(14) < 30（超卖）
2. 收盘价刺破布林带下轨（close < LowerBand）
3. 出现反转 K 线形态（锤子线 / 看涨吞没）
4. 做多，SL = 入场价 - 1%，TP = 布林带中轨

### 布林带参数
- 中轨：20 日均线（SMA20）
- 带宽：2 倍标准差

### 过滤条件
- 当前持仓方向与信号方向一致时，忽略反向信号（不做反向）
- 距上次平仓不足 30 分钟内不接新信号

## 信号数据结构

策略引擎输出以下信号字典，供 Runner 使用：

```python
signal = {
    "strategy": "EMA20_BREAKOUT",   # 策略名称
    "symbol": "BTCUSDT",            # 交易对
    "direction": "long",            # long / short
    "entry_price": 61000.0,         # 参考入场价
    "sl_price": 59700.0,            # 止损价
    "tp_price": 63500.0,            # 止盈价（SL * 1.5）
    "leverage": 5,                  # 建议杠杆
    "size": 1.0,                    # 合约张数（暂定）
    "confidence": 0.85,             # 信号置信度 0.0-1.0
    "reason": "EMA20上穿 + 放量突破", # 触发原因
    "kline_time": "2026-06-28T21:45:00Z",  # 信号K线时间
}
```

## 信号状态机

```
  [无持仓]
      │
      ▼
  检查 15m K线走完？
      │否 → 跳过
      │是
      ▼
  检查过滤条件
      │任一触发 → 跳过
      │全部通过
      ▼
  EMA20 方向？
   上翘 → LONG 信号
   下拐 → SHORT 信号
      │
      ▼
  风控计算
   ├─ 计算最大 size（账户 2% 风险）
   ├─ 计算 SL（K线最低/最高 ±0.5%）
   └─ 计算 TP（SL距离 × 1.5）
      │
      ▼
  [输出 signal dict] ──→ Runner 执行下单
```
